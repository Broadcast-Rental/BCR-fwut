Python is required to run esptool (which is bundled with this app). Follow these steps to install Python on your system.

1 Check if Python is already installed

First, let's check if Python is already installed on your system:

python3 --version

If you see a version number (like "Python 3.9.7"), Python is already installed! You can skip to step 3.

2 Install Python

macOS:

Option A - Using Homebrew (Recommended):

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install python

Option B - Download from python.org:

Download Python for macOS

Windows:

Download from python.org:

Download Python for Windows

Important: During installation, make sure to check "Add Python to PATH"

Linux (Ubuntu/Debian):

sudo apt update
sudo apt install python3 python3-pip
3 Verify Installation

After installation, verify Python is working:

python3 --version

You should see something like "Python 3.9.7" or similar.

4 Test the Firmware Uploader

Now go back to the Firmware Uploader app and try flashing firmware again. The app should now be able to use the bundled esptool with your Python installation.

Note: esptool is already bundled with this app, so you don't need to install it separately!

? Troubleshooting

If Python is still not found:

  • Windows: Restart your computer after installing Python
  • macOS: Try using python instead of python3
  • All platforms: Make sure Python is in your system PATH

Alternative commands to try:

python --version
/usr/bin/python3 --version
/usr/local/bin/python3 --version
