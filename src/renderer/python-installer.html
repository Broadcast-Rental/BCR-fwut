Python Installation Required

Python is needed to run esptool (bundled with this app). We can help you install it automatically or guide you through manual installation.

Manual Installation Guide

macOS:

1. Install Homebrew (if not already installed):

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. Install Python:

brew install python

3. Restart this app and try again.

Windows:

1. Download Python from the official website:

https://www.python.org/downloads/windows/

2. During installation, make sure to check "Add Python to PATH"

3. Restart this app and try again.

Linux (Ubuntu/Debian):

Install Python using your package manager:

sudo apt update && sudo apt install python3 python3-pip

Then restart this app and try again.
